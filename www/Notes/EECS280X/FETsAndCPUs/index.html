EECS 280X - FETs and CPUs
This section describes the essential low-level, foundational concepts.
Most important concepts (i.e. Test material) marked with ‘ * ‘.

Notes for EECS 280X (Programming and Data Structures) @ The University of Michigan.

Controllable Switches
  • The foundation of computers.
  • The switch itself is an abstraction, implementations vary.
    • Silicon is one such implementation.
Logic Gates
  • Critically important, covered in detail in Discrete Math.
  • Basic Gates:
    • not(a)
    • and(a,b) - TRUE if both a and b are true.
    • or(a, b) - TRUE if either a or b are true.
    • xor(a, b) - TRUE if either a or b are true, but not both.
  • More complicated gates can be generated by combining the basic gates.

ALU *
  • ALUs are the main instruction sets a CPU possesses.
    • | (bitwise OR), & (bitwise AND), ^ (bitwise XOR), >> (shift right), << (shift left)
    • Instructions simply stored at simple addresses.

Data Types (32 bit machine) *
  • Fundamental unit: Bit
  • nibble -> 4 bits
  • byte -> 8 bits
    • A char is (generally) 8 bits
  • short -> 2 bytes (16 bits)
  • int -> 4 bytes (32 bits)

Bases *
  • Binary (Base 2)
  • Hexadecimal (Base 16)
  • Decimal (Base 10)

Memories
  • Register File
    • Very, very small. 1 cycle latency.
  • Caches
    • Bigger than Register Files, but still very small.
    • ~32 kB, 3-10 cycle latency.
  • Main Memory
    • Very large, generally around a GB.
    • Hundreds of cycles latency.
