Identity Manager Improvement
Clicking the "+" button simply adds a new tab, allowing the user to fill out new details. All tabs other than the selected tab behave similar to the mail window tab strip. That is, every tab other than the default tab has a `min-width` and a `max-width`, so we can attempt to see most of the name. In addition, we should expand any tab on hover so the user can see the entire name.

This new tab format encourages the use of multiple identities and increases discoverability. We will share this tab strip across ALL accounts, so the identities are no longer account-dependent.
